## ğŸš€ Quick Start

### Setup for GitHub Pages

1. **Create a new repository** on GitHub named `username.github.io` (replace `username` with your GitHub username)

2. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/username/username.github.io.git
   cd username.github.io
   ```

3. **Add your files**:
   - Copy all the files from this portfolio into your repository
   - Add your profile picture and project images to the `assets` folder

4. **Commit and push**:
   ```bash
   git add .
   git commit -m "Initial portfolio setup"
   git push origin main
   ```

5. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click on "Settings"
   - Scroll down to "Pages" section
   - Under "Source", select "main" branch
   - Click "Save"

6. Your site will be live at `https://username.github.io` in a few minutes!

## ğŸ“ Customization Guide

### 1. Personal Information

Edit `index.html` to update:
- Your name (search for "Your Name" and replace)
- Email address
- Profile title/tagline
- About me section
- LinkedIn and GitHub links

### 2. Skills

Update the skills section in `index.html`:
- Change skill names
- Adjust skill percentages by modifying the `style="width: XX%"` attribute
- Add or remove skills as needed

### 3. Projects

For each project card in `index.html`:
- Update project title
- Change project description
- Add GitHub repository links
- Add project screenshots to `assets` folder and update image paths

### 4. Work Experience

Edit the timeline section:
- Update job titles
- Change dates
- Modify descriptions
- Add or remove timeline items

### 5. Images

Add your images to the `assets` folder:
- `profile.jpg` - Your profile picture
- `project1.jpg`, `project2.jpg`, etc. - Project screenshots
- Update image paths in `index.html` to match your filenames

### 6. Resume

- Add your resume PDF to the `assets` folder
- Update the resume links in the navigation and about section

### 7. Colors and Styling

Edit `style.css` to customize:
- Primary color: Change `--primary-color` in the `:root` section
- Background gradients in the hero section
- Fonts, spacing, and other design elements
